# E-Commerce Sales SQL Analysis

## Project Overview
This project analyzes E-Commerce sales data using MySQL to identify key sales trends and business insights.

## Tools Used
- MySQL
- MySQL Workbench
- SQL

## Analysis Performed
- Total Sales
- Total Orders
- Total Customers
- Sales by Category
- Sales by Region
- Top 10 Products by Sales
- Monthly Sales Trend
- Top 10 Customers by Sales
- Sales by Customer Segment
- Yearly Sales Analysis

## SQL Concepts Used
- SUM()
- COUNT()
- DISTINCT
- GROUP BY
- ORDER BY
- LIMIT
- STR_TO_DATE()
- DATE_FORMAT()
- YEAR()

## Project File
The complete SQL queries are available in `E-Commerce_Sales_Analysis.sql`.
